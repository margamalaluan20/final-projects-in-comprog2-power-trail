# Power-Up Run Treasure Trail 🏃💨💎

A dynamic 2D arcade-style game built entirely in Python using the Pygame library. This code demonstrates object-oriented programming principles, real-time event handling, and multimedia integration through an engaging "run and collect" gameplay loop.

## 🚀 Tech Stack

* **Programming Language:** Python 3
* **Game Engine / Library:** Pygame
* **Core Concepts:** Object-Oriented Programming (OOP), Collision Detection, Game Loop Mechanics

## ✨ Key Features

* **Dynamic Gameplay:** A continuous running experience where players navigate through obstacles and collect treasures.
* **Power-Up Mechanics:** Special items that temporarily boost the player's abilities or alter gameplay dynamics.
* **Custom Object Classes:** Modular code structure utilizing dedicated classes for game entities (e.g., players, obstacles, and items) found in `gameObject.py`.

## 📁 Repository Structure

* `main.py` - The primary entry point that initializes the game window and application state.
* `game.py` - Contains the core game loop, event handling, and rendering logic.
* `gameObject.py` - Defines the classes and behaviors for all interactive entities on screen.
